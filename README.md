## Install di Termux/Vps
```cmd
apt install install git nodejs ffmpeg libwebp
git clone https://github.com/IAmAdyy/Selfbot
cd Selfbot
npm i
node .
```
##### Note : If `apt install libwebp` didn't work
##### Use `apt install webp`
